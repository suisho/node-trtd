# trtd
Html table rendere that from js object


# install
```
npm install trtd
```

# usage
```js
var trtd = require("trtd")
var html = trtd({ head : "object"}, { body : "object"})
```

# API

## trtd( head, [body , [ opt ] ] )
- `head`
    - `Array` or `Object` or `undefined`
    - Rendered as head.
